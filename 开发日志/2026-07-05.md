## pointless-piglins 模组 — 从零到发布

### 项目概要
- Fabric mod for Minecraft 26.1.2, 移除自然生成僵尸猪灵的金矛
- 项目路径：`/Users/yepspoon/.openclaw/workspace/minecraft-mods/pointless-piglins`
- Mod ID: `pointless-piglins`
- JDK 25 (Adoptium at `/Library/Java/JavaVirtualMachines/jdk-25.jdk/Contents/Home`)

### 26.1.2 的类/API变化
- `ZombifiedPiglinEntity` → `ZombifiedPiglin`（去掉 Entity 后缀），包：`net.minecraft.world.entity.monster.zombie`（不是 piglin 包）
- `ZombifiedPiglin.populateDefaultEquipmentSlots()`：`nextInt(20)==0` → 5% 给 `GOLDEN_SPEAR`，否则 `GOLDEN_SWORD`
- `GameRules` 移到 `net.minecraft.world.level.gamerules.GameRules`
- `getGameRules()` 只在 `ServerLevel` 上可用
- Fabric API 的 gamerule API 变了：`GameRuleRegistry`/`GameRuleFactory` → `GameRuleBuilder`
- `Identifier.of()` → `Identifier.fromNamespaceAndPath()`

### 构建系统决策
- **放弃** buildscript + Loom SNAPSHOT 路线（ASM兼容问题，mappings各种坑）
- **采用** plugins DSL + **稳定版 Loom 1.17.13**（plugins DSL + pluginManagement 方式）
- **无需**自定义 mappings（稳定版 Loom 自动处理好了）
- 26.1.2 没有正式的 Yarn 映射，intermediary 只有 stub
- 但使用 Mojang 官方 mapping 名在源码中直接写类名可以正常编译

### gamerule
- ID: `pointless-piglins:disableGoldenSpear`
- 类型：Boolean，默认 true
- Category: MOBS
- true → 自然生成的僵尸猪灵不带金矛
- false → 原版行为
